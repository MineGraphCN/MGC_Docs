Iris 等效 OptiFine 功能的模组

这篇文档是近期新增的文档。

资料来源： Fabric Wiki - OptiFine alternatives

性能／兼容性

Sodium (Modrinth)

Iris 运行的必需品。

Indium (CurseForge) (Modrinth)

确保 Sodium 和某些模组兼容性的必需品。

缩放

Zoomify (CurseForge) (Modrinth)

有大量可自定义选项的缩放模组。

Logical Zoom (CurseForge) (Modrinth)

简单的缩放功能模组。

WI Zoom (CurseForge)

来自 Wurst 端的缩放模组。

Fabricated Zoomer (CurseForge)

OK Zoomer 的 Fabric 移植版。

资源包

自定义实体模型（CEM）

Entity Model Features (CurseForge) (Modrinth)

在 Forge 和 Fabric 上用以替代 OptiFine 的 CEM 模组。

Custom Entity Models (CurseForge) (Modrinth)

基于 OptiFine 格式的 CEM，旨在实现与 OptiFine 对等的 CEM 等功能。

发光／自定义／随机实体纹理

Entity Texture Features (CurseForge) (Modrinth)

添加了许多实体纹理特性，包括实体和玩家皮肤特性。

自定义物品纹理（CIT）

CIT Resewn (CurseForge) (Modrinth)

MCPatcher CIT 脱离 OptiFine 在 Fabric 上的独立模组。

Chime (CurseForge) (Modrinth)

和 MCPatcher / OptiFine 不兼容的 CIT 系统。

连接纹理（CTM）／自发光纹理

Continuity (CurseForge) (Modrinth)

为所有 OptiFine 格式的资源包提供 CTM 支持，同时还支持 OptiFine 原版自发光纹理。内置了一套连接纹理和一套玻璃剔除修复纹理。

动态纹理

MoreMcmeta (CurseForge) (Modrinth)

支持使用 json 和 OptiFine 文件给几乎所有纹理添加动画。提供纹理配置 API，并可以通过添加 附属模组 获得自发光纹理支持。

Animatica (CurseForge) (Modrinth)

支持 OptiFine / MCPatcher 格式动态纹理的模组。

自定义颜色

Polytone (CurseForge) (Modrinth)

自定义地图颜色、方块颜色、颜色映射图、群系颜色、染料颜色和方块音效。支持 OptiFine 格式。

Vanadium (Modrinth)

一个简单的实现资源包缓存和混合自定义颜色的模组。支持 OptiFine 格式。

Colormatic (CurseForge) (Modrinth)

和 Fabric 原本的自定义颜色模块兼容的独立实现模组。与 OptiFine 格式完全兼容，并添加了额外功能。

自定义天空

FabricSkyboxes (CurseForge) (Modrinth)

使用模组自定义的格式实现像 OptiFine 自定义天空一样的功能。

FabricSkyboxes Interop (CurseForge) (Modrinth)

支持 MCPatcher / OptiFine 格式的自定义天空，会自动将 OptiFine 自定义天空格式转换到 FabricSkyboxes 格式。两个模组搭配使用获得最好的效果。

动态光源

LambDynamicLights (CurseForge) (Modrinth)

让实体持有的光源和发光的实体本身产生光照。可能存在性能问题。

Mini Dynamic Lights (CurseForge) (Modrinth)

通过几乎最少的计算量实现快速动态光源的模组。

更好的草地

Fast Better Grass (资源包) (CurseForge) (Modrinth)

让草方块和相关的纹理侧面使用顶面纹理，可以与其他纹理包一起工作。 不像 OptiFine 在高品质下那样工作。

迷雾调整

Sodium Extra (CurseForge) (Modrinth)

Sodium 中没有的额外功能。

自定义启动屏幕

Puzzle (CurseForge) (Modrinth)

将上面的部分模组设置整合到一个屏幕里，并添加了资源包功能。

自定义 GUI

OptiGUI (CurseForge) (Modrinth)

自定义物品栏 GUI 和其他更多特性。
